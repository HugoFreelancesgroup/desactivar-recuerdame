Desactiva la casilla "Recuérdame" en el formulario de acceso de WordPress para mejorar la seguridad.

== Descripción ==
Este plugin oculta la opción "Recuérdame" que aparece en el formulario de login de WordPress, evitando que los usuarios puedan iniciar sesión con sesiones prolongadas. Es una mejora ligera de seguridad para sitios que desean minimizar riesgos.

== Instalación ==
1. Sube la carpeta `desactivar-recuerdame` al directorio `/wp-content/plugins/`.
2. Activa el plugin a través del menú "Plugins" en WordPress.
